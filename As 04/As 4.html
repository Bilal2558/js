“Rules for naming JS variables”
1 Variable names can only contain letters, numbers, dollar signs, and underscores. For example $my_1stVariable
2 Variables must begin with a dollar signs underscores or letters. For example $name, _name or name
3 Variable names are case sensitive
4 Variable names should not be JS keywords.
